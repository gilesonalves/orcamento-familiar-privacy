# Política de Privacidade – Orçamento Familiar

Esta Política de Privacidade descreve como o aplicativo **Orçamento Familiar** coleta, usa e protege as informações dos usuários.

## 1. Coleta de informações
O aplicativo coleta apenas as informações necessárias para o funcionamento do serviço, como:
- Dados de autenticação (login), fornecidos pelo próprio usuário.
- Informações financeiras inseridas manualmente pelo usuário (receitas e despesas).

Não coletamos dados sensíveis automaticamente, nem informações de localização, contatos ou dados do dispositivo.

## 2. Uso das informações
As informações coletadas são usadas exclusivamente para:
- Permitir o funcionamento do aplicativo.
- Armazenar e organizar receitas e despesas do usuário.
- Melhorar a experiência do usuário.

## 3. Compartilhamento de dados
Os dados **não são vendidos, compartilhados ou transferidos** para terceiros, exceto quando necessário para o funcionamento técnico do app (ex.: serviços de backend).

## 4. Armazenamento e segurança
Os dados são armazenados de forma segura utilizando serviços de backend confiáveis. Medidas de segurança são adotadas para proteger as informações contra acesso não autorizado.

## 5. Controle do usuário
O usuário pode:
- Visualizar, editar ou excluir seus dados a qualquer momento dentro do app.
- Solicitar a exclusão da conta e dos dados associados.

## 6. Crianças
O aplicativo **não é direcionado a crianças menores de 13 anos**.

## 7. Alterações nesta política
Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página ocasionalmente.
