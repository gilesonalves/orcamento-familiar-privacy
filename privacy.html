Política de Privacidade

O aplicativo Orçamento Familiar respeita a privacidade dos seus usuários.

Dados coletados

Coletamos apenas os dados necessários para o funcionamento do aplicativo, como:

  • Dados de autenticação (e-mail ou login social)
  • Informações financeiras inseridas pelo próprio usuário

Uso dos dados

Os dados são utilizados exclusivamente para:

  • Permitir o login do usuário
  • Gerenciar receitas e despesas
  • Melhorar a experiência no aplicativo

Compartilhamento

Não compartilhamos dados pessoais com terceiros.

Segurança

Os dados são transmitidos de forma segura (criptografia em trânsito) e armazenados de maneira protegida.

Exclusão de dados

O usuário pode solicitar a exclusão da conta e dos dados a qualquer momento.
